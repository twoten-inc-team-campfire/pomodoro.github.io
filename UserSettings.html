Constructor

new UserSettings(pomodoroLength, shortBreakLength, longBreakLength, numPomodorosInCycle, autoStartBreaks, autoStartPomodoros, displayPauseButton, displayCancelButton, displayFastForwardButton, displayTaskSelector)

Create a new UserSettings object using the default values established in the configuration file.
Parameters:
Name Type Default Description
pomodoroLength number 25 Length of the pomodoro cycle in whole minutes.
shortBreakLength number 5 Length of the short break in whole minutes.
longBreakLength number 20 Length of the long break in whole minutes.
numPomodorosInCycle number 4 The number of pomodoros in a complete cycle (before a long break)
autoStartBreaks boolean true Start breaks without user input immediately after finishing a pomodoro
autoStartPomodoros boolean true Start pomodoros without user input immediately after finishing a break.
displayPauseButton boolean true Whether to display the pause button on the home screen.
displayCancelButton boolean true Whether to display the cancel button on the home screen.
displayFastForwardButton boolean true Whether to display the fast forward button on the home screen.
displayTaskSelector boolean true Whether to display the task selector field on the home screen.
